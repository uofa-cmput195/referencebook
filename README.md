# Introduction to Principles and Techniques of Data Science 
This repository holds the Jupyter Book source for CMPUT 195: Introduction to Principles and Techniques of Data Science textbook.

## To make a change to the book and update `https://uofa-cmput195.github.io/referencebook/`

1. Get your copy of this repository:

   ```
   git clone https://github.com/uofa-cmput195/referencebook.git
   ```
2. Change the file you wish and commit it to the repository.
3. Push your change back to the `uofa-cmput195/referencebook` repository.
4. That's it a GitHub Action will build the book and deploy it to [uofa-cmput195.github.io/referencebook](https://uofa-cmput195.github.io/referencebook/)
